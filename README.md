# Taller Polimorfismo
Se crean 3 clases 
clase FiguraGeometrica que actua como super clase y contiene sus metodos get, set & str
clase cuadrado que recibe un solo valor y los asigna a ambos atributos usando setters y sobreescribe los metodos area y perimetro
clase rectangulo que sobre escribe los metodos area y perimetro
el archivo main en el que se crean y prueban el funcionamiento de los metodos creados
